FAQ Rote Games

'Rote Games' is an app that aims to help enhance your memory through games where you must remember and repeat sequences of elements.

The app has 4 different game modes:

  1. Classic mode: the user has to repeat the elements in the same order in which they appear.
  2. Reverse mode: the user has to repeat the elements in reverse order i.e. from the last to the first element of the sequence.
  3. Random mode: each round, the game generates a different sequence. The user has to repeat it in the same order in which the elements appear.
  4. Mixed mode: the user has to repeat the sequence in the order specified at the beginning. Every 5 rounds, a new sequence is generated and the order to enter the elements may change.

You can unlock each game mode via In-app purchases.
